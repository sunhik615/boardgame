
# 보드게임 이미지 추가 가이드

이미지가 없는 게임들에 이미지를 추가하는 방법입니다.

## 1. 이미지 누락 목록 확인
`missing_images.md` 파일을 열면 이미지가 없는 게임들의 목록과 ID를 확인할 수 있습니다.

## 2. 이미지 파일 준비
1. [BoardGameGeek](https://boardgamegeek.com/) 또는 구글 검색을 통해 게임의 박스 커버 이미지를 다운로드합니다.
2. **중요**: 이미지 파일의 이름을 게임의 **ID** 또는 **게임 타이틀**(보이는 그대로)과 똑같이 변경해주세요.
   - 예: 게임 ID가 `yamatai`라면 파일명은 `yamatai.jpg`, 게임 타이틀이 `3초 트라이`라면 `3초 트라이.jpg` 모두 가능합니다.
   - 확장자는 `.jpg`, `.png`, `.webp`, `.jpeg` 모두 가능합니다.

## 3. 이미지 파일 이동
준비된 이미지 파일들을 아래 폴더에 넣어주세요:
`assets/images/games/`

## 4. 자동 업데이트 실행
이미지를 폴더에 넣은 후, 터미널에서 다음 명령어를 실행하면 `games-data.js`에 자동으로 이미지가 연결됩니다.

```bash
node update_images.js
```

명령어가 성공적으로 실행되면 "games-data.js has been updated!" 메시지가 뜹니다.
이제 웹페이지를 새로고침하면 이미지가 적용된 것을 확인할 수 있습니다.
